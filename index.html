wp-pullquotes

Provides a repository of linkable pull quotes for your Wordpress site

Download .zip Download .tar.gz View on GitHub

Do what now?

  • Download and unzip to your wp-content/plugins directory
  • Visit the installed plugins area of your dashboard and activate the plugin
  • You will now see a "Pull Quotes" menu item where you can add quotes galore
  • Use the Weight option (1-10) to bias how often a particluar quote will display
  • Hey look! You can see how many times each quote is displayed
  • Visit the Appearance > Widgets area to add the Pull Quote widget to you sidebar. Hey look! You can adjst the speed ar which the quotes is typed
